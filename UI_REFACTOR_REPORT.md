- **宇宙星空主题**: 深色渐变背景 (#0a0e27 → #1a1f3a → #0f172a)

## 🎯 实现技巧

### 1. 宇宙背景实现
```css
/* 三层渐变背景 */
background: linear-gradient(
  135deg, 
  #0a0e27 0%, 
  #1a1f3a 50%, 
  #0f172a 100%
);

/* 光晕动画 */
.cosmic-bg::before {
  background: radial-gradient(
    circle, 
    rgba(0, 217, 255, 0.3) 0%, 
    transparent 70%
  );
  filter: blur(120rpx);
  animation: cosmicPulse 8s infinite;
}
```

### 2. 玻璃拟态实现
```css
.glass-card {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(20rpx);
  border: 1rpx solid rgba(255, 255, 255, 0.1);
}
```

### 3. 星星动画实现
```css
.star {
  width: 4rpx;
  height: 4rpx;
  background: white;
  border-radius: 50%;
  animation: twinkle 2s infinite ease-in-out;
}

@keyframes twinkle {
  0%, 100% { opacity: 0.3; transform: scale(1); }
  50% { opacity: 1; transform: scale(1.2); }
}
```

### 4. 渐变文字实现
```css
.title {
  background: linear-gradient(
    135deg, 
    #00d9ff 0%, 
    #b794f6 50%, 
    #ec4899 100%
  );
  background-clip: text;
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

## 💡 注意事项

### 小程序限制
1. **不支持 `:hover`**: 用 `:active` 替代
2. **backdrop-filter 兼容性**: iOS 10+ 支持，Android 部分支持
3. **动画性能**: 避免过多同时动画
4. **层级限制**: `z-index` 在某些组件无效

### 适配建议
1. **安全区域**: 使用 `env(safe-area-inset-bottom)`
2. **深色模式**: 可考虑添加深色模式适配
3. **性能**: 星星数量控制在 20-30 个
4. **包大小**: 背景图可考虑压缩
