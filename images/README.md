# 图标资源说明

本目录存放小程序所需的 TabBar 图标。

## 所需图标

请准备以下 6 个 PNG 图标文件（尺寸：81x81 像素）：

### 学习 Tab
- `learn.png` - 学习图标（未选中）
- `learn-active.png` - 学习图标（选中，绿色 #07c160）

### 测验 Tab
- `quiz.png` - 测验图标（未选中）
- `quiz-active.png` - 测验图标（选中，绿色 #07c160）

### 收藏 Tab
- `collection.png` - 收藏图标（未选中）
- `collection-active.png` - 收藏图标（选中，绿色 #07c160）

## 图标建议

可以使用以下资源生成图标：

1. **iconfont** (https://www.iconfont.cn/) - 阿里巴巴矢量图标库
2. **Icons8** (https://icons8.com/) - 免费图标网站
3. **Flaticon** (https://www.flaticon.com/) - 矢量图标搜索

## 临时方案

如果暂时没有图标，可以使用微信开发者工具默认图标，或暂时注释掉 app.json 中的 tabBar 配置。

## 图标风格建议

- 简洁线条风格
- 白色背景或透明背景
- 未选中：灰色 (#999)
- 选中：绿色 (#07c160)
