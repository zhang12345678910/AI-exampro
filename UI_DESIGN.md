## 🎨 设计规范

### 图标风格
- **风格**: 线性图标 (Linear Icons)
- **描边**: 2px 圆角线条
- **颜色**: 
  - 未选中：#999999
  - 选中：#1E88E5 (科技蓝)
- **背景**: 透明

### 配色方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| **主色调** | #1E88E5 | 导航栏、主按钮、选中状态 |
| **辅助色** | #E3F2FD | 浅色背景、卡片底色 |
| **强调色** | #FF9800 | 打卡标识、重要提示 |
| **成功色** | #4CAF50 | 完成状态、正确提示 |
| **警告色** | #FFC107 | 注意提示 |
| **错误色** | #F44336 | 错误提示 |
| **文字主色** | #333333 | 主要文字 |
| **文字辅色** | #666666 | 次要文字 |
| **文字提示** | #999999 | 提示文字 |
| **背景色** | #F5F7FA | 页面背景 |
| **卡片背景** | #FFFFFF | 卡片背景 |

### 字体规范

| 用途 | 字号 | 字重 | 行高 | 颜色 |
|------|------|------|------|------|
| **大标题** | 36rpx | Bold | 1.4 | #333 |

| **卡片标题** | 28rpx | Bold | 1.4 | #333 |
| **正文** | 28rpx | Regular | 1.5 | #333 |
| **辅助文字** | 24rpx | Regular | 1.5 | #666 |
| **提示文字** | 24rpx | Regular | 1.5 | #999 |

### 间距规范

| 类型 | 数值 | 说明 |
|------|------|------|
| **页面边距** | 24rpx | 左右边距 |
| **卡片内边距** | 24rpx | 卡片内容padding |
| **元素间距** | 16rpx | 元素之间间距 |

## 📤 切图输出规范

### 输出格式
- **图标**: PNG @2x, @3x
- **卡片**: PNG @2x
- **勋章**: PNG @2x, @3x
- **背景图**: JPG (压缩质量 80%)

### 命名规范
```
{模块名}-{元素名}-{状态}.{格式}
示例:
- tab-home.png
- tab-home-active.png
- badge-001.png
- card-template-1.png
```

