## 📋 升级概述

### 当前项目状态
- ✅ 已完成：基础框架、4 个核心页面、术语库、收藏功能、学习进度追踪
- ⏳ 待完成：测验功能、复习提醒、分享功能、分级学习体系

### 新 PRD 核心变化

| 维度 | 原方案 | 新 PRD 要求 | 升级重点 |
|------|--------|-------------|----------|
| **功能模块** | 4 个模块（学习、测验、收藏） | 6 个模块（首页、词典、学习、我的、工具） | 新增首页模块、我的模块、工具模块 |

| **工具能力** | 基础分享 | 名词卡片生成（多模板）、多场景分享 | 内容创作工具 |

## 📱 页面设计详细方案

### 1. 首页 (index)

**页面路径**: `/pages/index/index`

**功能模块**:
```
┌─────────────────────────────────┐
│  🔍 搜索框 (关键词/拼音/联想)    │
├─────────────────────────────────┤
│  📅 每日一词卡片                 │
│  ┌───────────────────────────┐  │
│  │ 机器学习                  │  │
│  │ AI 让计算机从数据中学习     │  │
│  │ [打卡] [收藏] [分享]       │  │
│  └───────────────────────────┘  │
├─────────────────────────────────┤
│  分类入口 (横向滑动)             │
│  [入门必备] [进阶提升] [...]     │
├─────────────────────────────────┤
│  🔥 热门名词推荐                 │
│  • 机器学习 [入门]              │
│  • 深度学习 [进阶]              │
│  • ... (10 个)                  │
└─────────────────────────────────┘
```

**核心交互**:
- 搜索框输入≥1 字符触发联想
- 历史记录最多 10 条，可删除/清空
- 每日一词每日 0 点自动更新
- 打卡后显示"今日已打卡"

## 📈 数据指标

### 核心指标

| 指标 | 目标值 | 计算方式 |
|------|--------|----------|
| 用户留存率 | ≥30% | 次日留存 |
| 日均查询量 | ≥5 次/用户 | 人均每日查询名词数 |
| 打卡率 | ≥40% | 每日打卡用户/活跃用户 |
| 测验参与率 | ≥25% | 参与测验用户/活跃用户 |

### 数据埋点

| 事件 | 触发时机 | 上报数据 |
|------|----------|----------|
| `search` | 用户搜索 | `{ keyword, resultCount }` |
| `view_term` | 查看名词详情 | `{ termId, difficulty, category }` |
| `collect_term` | 收藏名词 | `{ termId }` |
| `checkin` | 每日打卡 | `{ continuousDays }` |
| `quiz_start` | 开始测验 | `{ level, questionCount }` |
| `quiz_complete` | 完成测验 | `{ level, score, timeSpent }` |
| `share` | 分享内容 | `{ type, target }` |
