## 🎨 UI 设计规范

### 配色方案
```
主色调：#1E88E5 (科技蓝)
辅助色：#E3F2FD (浅蓝)
强调色：#FF9800 (橙色)
成功色：#4CAF50 (绿色)
警告色：#FFC107 (黄色)
错误色：#F44336 (红色)
```

### 字体规范
```
大标题：36rpx Bold
页面标题：32rpx Bold
卡片标题：28rpx Bold
正文：28rpx Regular
辅助文字：24rpx Regular
```

### 间距规范
```

元素间距：16rpx
大间距：32rpx
卡片圆角：16rpx
```

## ⚠️ 风险与应对

### 风险 1: 术语库扩展工作量大
**应对**: 
- 分批扩展（200 → 500 → 1000）
- 使用 AI 辅助生成
- 社区贡献

### 风险 2: 开发周期紧张
**应对**:
- 优先保证 P0 功能
- P1 功能分迭代
- 复用现有组件

### 风险 3: 性能问题
**应对**:
- 数据分页加载
- 图片懒加载
- 本地缓存优化
